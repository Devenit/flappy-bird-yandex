Проект Pygame Flappy Bird

Flappy Bird - игра, в которой игрок должен контролировать полёт птицы между рядами зеленых труб.
При запуске игры вас будет встречать две сложности: Standard и Speed. Отличие в них в скорости игры.
У каждой сложности есть свой счетчик и рекорд.
Выйти из игры можно нажав ESCAPE. Контролировать полёт можно при помощи клавиш мыши.

Вы проиграете, если коснетесь трубы. Вас встретит меню, где можно начать заново или уйти в главное меню.
Цель: набрать больше всего очков.
